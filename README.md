# End-To-End Supplier Product Scrape Service

Bu proje bazı tektsil tedarik firma sitelerinden ürün kodu yoluyla ürün bilgilerini otomatik olarak çekmeye yarayan araçları içerir.


### Mimari

Backend uygulaması Flask, desktop frontend uygulaması PyQt6, mobil uygulama Flutter ile tasarlanmıştır.
Desktop app hem uzaktaki sunucu yoluyla hem de lokal cihaz yoluyla çalışabilmektedir. Proje şeması bu görselde belirtilmiştir.
Tek bir noktadan farklı tedarikçi seçimleriyle veriler çekilir, istenirse İkas ürün formatınna otomatik olarak dönüştürülür veya ham olarak alınabilir.
